Welcome to the world's greatest comics API!

The Marvel Comics API is a tool to help developers everywhere create amazing, uncanny and incredible web sites and applications using data from the 70-plus years of the Marvel age of comics.

Here's the short version of what you need to do:

  1. Sign up: Get an API key
  2. Be a good API citizen: read, understand, and abide by the terms of use for the Marvel Comics API
  3. Link back: observe the attribution and linking guidelines when displaying data from the API
  4. Keep in touch: tell us about what you're building and talk to other developers on our community page
  5. Build cool stuff
